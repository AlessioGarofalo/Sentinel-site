Sovereign Mode

Your intelligence
stays yours.

Most intelligence platforms send your queries, documents, and analysis to cloud servers. Sentinel is different. In Sovereign Mode, every byte stays on your hardware.

✓
Local LLM via Ollama — Qwen 3 32B runs entirely on-device. No OpenAI account required.
✓
Local Speech — Whisper STT and Kokoro TTS run offline. Voice never leaves your machine.
✓
Encrypted vault — All data lives in a VeraCrypt/BitLocker container. Unlock before launch.
✓
Localhost-only binding — Server refuses non-local connections by design.
✓
Air-gap ready — Public signal feeds are optional. Core analysis works fully disconnected.

FAQ

Common questions.

In Sovereign Mode, Sentinel runs entirely offline. Live signal feeds (news, AIS, weather, etc.) require internet, but AI analysis via Qwen, voice, and the private knowledge base work fully air-gapped.
Sentinel currently runs on Windows 10 and Windows 11 only. No installation is required, the portable build bundles Python, Chromium, and Ollama. macOS and Linux builds are on the roadmap.
No. In Sovereign Mode, Sentinel runs Qwen 3 32B locally via Ollama, no OpenAI account or API key is required. If you prefer OpenAI, you connect your own key and it is stored locally in your encrypted vault.
For OpenAI mode, any modern laptop works. For local Qwen 3 32B, we recommend 32GB RAM and an NVIDIA GPU (16GB VRAM+) for best performance.
The Sovereign Edition ships with a VeraCrypt-compatible vault structure. We recommend enabling full-disk encryption (BitLocker or VeraCrypt) before first use. The launcher integrates with an encrypted container unlock step.
The Solo licence (€699) is bound to one PC, it will not run on a different machine. The Sovereign Edition (€999) is bound to the physical drive, not the PC — plug it into any Windows machine and it works. You can move between computers freely; you cannot clone the drive and run two instances simultaneously.
