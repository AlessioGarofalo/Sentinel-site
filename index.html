Path-Aware Trade Performance Resolver
Paper trade outcomes are now resolved against actual OHLC bar history rather than a single latest price snapshot. The resolver walks each daily bar between entry and evaluation, checking whether the stop-loss or take-profit was touched first. When both levels appear inside the same bar, the stop is assumed to have triggered first — a conservative default that prevents overstated P&L in volatile sessions.
